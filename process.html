Learning process

After completing the HTML courses at Code Academy, we had a rudimentary understanding of coding. We started by brainstorming the content, focusing on what and where to write because we didn't know much about CSS at the time. As we delved deeper into CSS, we discovered the intricacies of website design as we tackled the challenges of element placement and color selection. It was difficult to create the ideal color contrast and design combination. We also finished JS. Besides HTML and CSS, we also learned JavaScript.

GitHub

We first tried to understand the logic of github and how to communicate with each other there. We guided each other and helped each other by demonstrating our work there. Having all the data on github and being able to upload it has been very helpful in terms of data protection and also in terms of being able to easily communicate with each other
